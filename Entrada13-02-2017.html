Para comenzar debemos aprender a diferenciar entre INTERNET y la WEB ya que son 2 cosas distintas.
Internet es una conjunto de redes de comunicación interconectadas y la WEB es un sistema de distribución de documentos de hipertexto o hipermedios accesibles vía internet.
Para conocer más acerca de cada uno te invito a seguir los siguientes enlaces donde encontraras mayor información y mas detallada.
INTERNET WEB
Despues de haber entendio y conociodo un poco mas acerca de la historia de la web explicaremos como funciona una pagina web y sus diferentes tipos que existen.

Esxisten 2 tipos de paginas:

  • PGINAS WEB ESTATICAS
  • PGINAS WEB DINAMICAS

Para mostrar mas facil como funciona los dos tipos de paginas web motraremos el procedimiento de Solicitu/Respuesta.

Procedimiento solicitud/respuesta del cliente/servidor WEB ESTATICA

  1. Usuario ecribe URL
  2. Navegador envia la URL al DNS
  3. El DNS regresa la IP correspondiente a la URL
  4. Navegador solicita directamente la pagina utilizando la IP
  5. Servidor recibe la solicitud de la pagina (index.html)
  6. El servidor busca la pagina en el disco duro
  7. El servidor regresa la pagina al navegador (html)
  8. Eleinte reciibe el despliege de la pagina en htm

Procedimiento solicitud/respuesta del cliente/servidor WEB DINAMICA

  1. El usuario introduce el URL (Navegador)
  2. Se busca la IP (Internet)
  3. solicita la pagina ya con la IP (Navegador)
  4. Recibe la solicitud de la pagina (servidor)
  5. Obtiene la pagina (HDD del servidor) index.php
  6. Contiene PHP y la envia a PHP
  7. Procesa PHP (procesador PHP)
  8. Regresa la pagina (Servidor)
  9. Despliegue en el navegador

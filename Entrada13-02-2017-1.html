Estructura semantica de una pagina web - Clase 31/01/2017

HTML Semántico es el uso de las etiquetas HTML para reforzar la semántica, o el significado, de la información en las páginas web más que simplemente redefinir su forma de presentación (apariencia). El HTML semántico es procesado por los navegadores web regulares así como por muchos otros agentes de usuarios. Las CSS son usadas para sugerir la presentación de la página a usuarios humanos. Es muy importante una buena estructura semantica de una pagina web.

Partes de estructura:

  • Encabezado <header>
  • Navegación <nav>
  • Contenido principa <main>
  • Sección <section>
  • Articulo <articulo>
  • Barra Lateral <aside>
  • Pie de pagina <footer>
  • Divisiones <div>

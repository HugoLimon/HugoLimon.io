Arreglos en JavaScript

En JavaScript existen diferentes maneras de crear un arreglo, a continución motraremos como hacerlo como también su distintos métodos.

Creación

  • var arr = new Array(); arreglo vacío
  • var arr = []; arreglo vacío
  • var arr = new Array('cadena', 123, true)
  • var arr = ['cadena', 123, true]
  • var matriz = [[1,2,3],[4,3,2]]

Métodos

  • .concat(b) devuelve otro arreglo
  • .join() une todos los elementos de un arreglo en una cadena
  • .push() agrega un elemnto al final del arreglo
  • .pop() elimina el ultimo elemento del arreglo
  • .reverse() Regresa un arreglo con los elementos en orden invertido
  • .Sort() ordenan un arreglo de forma ascendente solo con cadenas.

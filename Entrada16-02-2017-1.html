Imagenes y Enlaces en HTML - Clase 30/01/2017

Unos de los elementos más usados en las pagínas web son las imagenes y los enlaces, la mayora de las pagínas web usan estos 2 elementos estas vez se mostraran las etiquetas principales y atributos para estos.

Imagenes

Etiqueta <img>

  • <img src="nombre del archivo">
  • <img height="altura">
  • <img width="ancho">
  • <img alt="texto alternativo">

Enlaces

Etiqueta <a> ejemplo:

  • <a> https://www.google.es </a>

Introducción CSS - Clase 7/02/2017

Hojas de estilo en cascada (o CSS, siglas en inglés de Cascading Stylesheets) es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado . Es muy usado para establecer el diseño visual de las páginas web, e interfaces de usuario escritas en HTML o XHTML; el lenguaje puede ser aplicado a cualquier documento XML, incluyendo XHTML, SVG, XUL, RSS, etcétera. También permite aplicar estilos no visuales, como las hojas de estilo auditivas.
https://es.wikipedia.org/wiki/Hoja_de_estilos_en_cascada

Reglas

Las reglas permiten cambiar la apariencia (presentación) de los elementos de HTML.
Sintaxis general Selector{declaraciones}. Las declaraciones se componen de una propiedad: valor.

Ejemplo

selector { propiedad: valor }
{color: blue}
p { font-size: 200% }

Tipos de estilos

  • Por default
  • Del usuario (archivo local)
  • Hoja de estilo externa (utiliza la etiqueta para hacer referencia al archivo)
  • Estilo interno (utiliza la etiqueta <style>)
  • Estilo en linea (se asigna la propiedad y valor como atributo del elemento)

Selectores CSS

  • Selectorde tipo
    • Selecciona un tipo de elementop HTML y aplica el estilo a todos los elementos delmismo tipo.
      Ej. p{texto-aling:justify;}
  • Selector descendente
    • Aplica el estio a elementos que estan contenidos dentro de otros elementos
      p b{color:red;}
      li b {color:green;}
  • Selector hijo
    • Similar al decendente pero mas restictivo aplica el esticlo solamente a los elementos que sean hijos directos de otro elemento.
      Ej. p>b {color:blue;}
  • Selector por id
    • Un id es un atributo de un elemnto html y se utiliza para aplicar un estilo de manera particular.
      Ejemplo: <p id="introduccion">
      Un selector por id utiliza el simbolo #
      #introduccion {color:blue;}
  • Selector por clase
    • Aplicar estilo a un conjunto de elementos Ej. <p class="especial"> este es un parrafo especial</p >
      /* en CSS */
      .especial{color:red}
  • Selector universal *
    • Aplica el mismo estilo a todos los elementos de la pagina los estilos se heredan, si aplicas estilo un elemento, todos los elementos.
